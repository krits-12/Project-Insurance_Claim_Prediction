# Auto Insurance Claim Prediction

## Description
A machine learning project to predict the likelihood of an auto insurance claim using binary classification.

## Requirements
- Python 3.10+
- Install dependencies using: `pip install -r requirements.txt`

## How to Run
1. Clone this repository.
2. Install the requirements.
3. Open `notebooks/DGE301_AutoInsuranceClaim.ipynb` in Jupyter or Google Colab.
4. Run all cells in order.

## Dataset
- The dataset is located in the `/data` folder
